# 画面レイアウト設計

本ドキュメントは、画面遷移図（screen_transition_diagram.md）に基づく各画面のレイアウト設計をまとめたものです。

---

## ホーム画面

### 主な UI 要素

- アプリ名・ロゴ
- 「Google でログイン」ボタン
- 「利用規約」リンク
- 「マイページ」ボタン（ログイン済みの場合）

### ワイヤーフレーム（例）

```
+-----------------------------+
|        アプリ名・ロゴ        |
|-----------------------------|
| [Googleでログイン]           |
|-----------------------------|
| [利用規約]   [マイページ]    |
+-----------------------------+
```

---

## Google アカウントや X アカウントでログイン

### 主な UI 要素

- Google ログインボタン
- X（旧 Twitter）ログインボタン
- ログイン状態の表示

### ワイヤーフレーム（例）

```
+-----------------------------+
|      アカウントでログイン     |
|-----------------------------|
| [Googleでログイン]           |
| [Xでログイン]                |
+-----------------------------+
```

### 主な UI 要素

```

|-----------------------------|
|  [利用規約本文]             |

+-----------------------------+
```

## マイページ

- 「最新の投票履歴」ボタン
- 「過去の投票履歴」ボタン
- 「ログアウト」ボタン

| [最新の投票履歴]            |
